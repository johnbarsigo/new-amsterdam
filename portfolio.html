Multi-Vendor Food Ordering and Management System


Developed a RESTful full-stack ordering and vendor management system enabling menu publishing, customer ordering and transaction workflow management.


What I did:


  • Implemented the core functionality of the system, set up models and routes.
  • Assisted in connecting the frontend to the backend.
  • Optimized database queries for improved performance.
  • Conducted testing and debugging to ensure system stability.
  • Implemented a real-time customer favourites system based on customer feedback.
  • Applied authentication and authorization mechanisms to ensure secure access to the system by using tools such as :
    • JWT (JSON Web Tokens)
    • OAuth 2.0
    • Role-Based Access Control (RBAC)
    • Password Hashing
